# Музыкальный каталог популярной музыки

**Мобильное Android-приложение** для поиска, просмотра и прослушивания превью популярных музыкальных треков с поддержкой полноценного оффлайн-режима.

Приложение загружает актуальные чарты и данные о треках через официальный **Deezer API**, кэширует их в локальной базе Room и позволяет пользоваться каталогом даже без интернета.

Идеально подходит для меломанов, которые хотят быстро найти хит, посмотреть обложку, длительность, исполнителя — и слушать 30-секундные превью в любое время.

## Основные возможности

- Загрузка топ-чартов и популярных треков через Deezer API
- Полноценный поиск по названию трека
- Кэширование всех данных в локальной базе Room
- Полностью рабочий **оффлайн-режим** (после первой загрузки интернет не нужен)
- Отображение обложек альбомов, названия трека, исполнителя, длительности
- Прослушивание 30-секундных превью треков (если доступно в Deezer)
- Современный Material You интерфейс на Jetpack Compose

## Технологический стек

### Язык и платформа
- **Kotlin** 1.9+
- Android SDK (minSdk 24 — Android 7.0 Nougat, targetSdk 35)

### Архитектура
- MVVM + Clean Architecture
- Single Activity, Jetpack Navigation

### Основные библиотеки
| Назначение              | Библиотека                        |
|-------------------------|-----------------------------------|
| UI                      | Jetpack Compose + Material 3      |
| Локальная БД            | Room 2.6+                         |
| Сетевые запросы         | Retrofit 2 + Moshi/Gson           |
| Изображения             | Coil Compose                      |
| Асинхронность           | Kotlin Coroutines + Flow          |
| DI                      | Hilt (или Koin — уточни, если нужно) |
| API                     | Deezer Public API[](https://developers.deezer.com) |

## Системные требования

- Android 7.0 (API 24) и выше
- Минимум 2 ГБ оперативной памяти
- ~100 МБ свободного места (включая кэш обложек)
